### 数据结构：是用来计算机来存储、组织数据的方式

-     广义的说法：数据结构 = 数据存储 + 算法
-     狭义的说法：数据的存储
  基本概念：
  数据 -> 数据对象 -> 数据元素 -> 数据项

结构：数据元素之间的关系叫做结构

- 逻辑结构：反映数据元素之前的逻辑关系，如集合、线性结构（一维数组、栈、队列）、非线性结构（树、图、多维数组）
- 存储结构：数据结构在计算机中的表示，如顺序存储结构、链式存储结构、索引存储结构、散列存储结构
  线性是连续的，集合是离散的，树结构和图结构都是非线性结构

算法：对数据的操作

集合：

1. 集合中的数据没有先后顺序
2. 每个数据成员在数据中不能重复出现，仅且只出现一次， js 中的 set 就是集合

线性表：数据元素之间是一对一关系，也就是数据元素一个接一个的排列

1. 用来存放特定的某一类型的元素
2. 物理结构为顺序表和链表
   衍生结构：

- 栈
- 队列
- 串

树状结构：二叉树
二叉树的遍历：深度优先遍历、广度优先遍历
前序、中序、后序

内存与存储结构：
内存：一维线性结构
